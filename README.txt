ExemploLombokJSF

Feito para exemplificar a utilização do projeto Lombok (http://projectlombok.org) com o Java Server Faces (JSF).

A intenção é demonstrar como o código contido nas classes Java fica menor e de certo modo mais legível com as anotações @Getter e @Setter nos métodos.

Este repositório possuirá dois branchs sendo o master contendo o código antes de utilizar o Lombok e o branch lombok possuindo as refatorações para usufruir das anotações comentadas.
